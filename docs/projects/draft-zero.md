# Draft Zero Creative Suite

Full creative writing suite — analyze, write, and adapt across all formats.

## Overview

- **Tech:** Next.js, Haiku 4.5 (default), Sonnet 4.5 (quality), Opus (heavy)
- **Password:** draftzero2026
- **Status:** ✅ Complete

- Deep structure analysis for each format
- Chat + write modes
- Adaptation engine (screenplay ↔ novel ↔ stage play)
- Visual pipeline (screenplay → graphic novel via FLUX)

## Formats Supported

- Screenplays
- Novels
- Stage plays
- Short stories
- Graphic novel scripts
