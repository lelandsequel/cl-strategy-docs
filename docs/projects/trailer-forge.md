# Trailer Forge

Screenplay → AI-generated movie trailer.

## Overview

- **Tech:** Next.js, Haiku 4.5, Runway Gen-3/FLUX for video, Suno for music
- **Password:** trailerforge2026
- **Status:** ✅ Complete

## Features

- Screenplay parsing
- "Trailer moment" selection (hooks, conflicts, spectacle)
- Shot breakdown with video prompts
- AI video generation
- Audio pipeline (music + SFX)
- Composition into final trailer
