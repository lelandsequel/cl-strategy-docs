## Adding New Docs

1. Add project folder to `docs/projects/`
2. Update `mkdocs.yml` nav section
3. Run `mkdocs serve` to preview
4. Run `mkdocs build` to generate static site

## Tech Stack

- **Frontend:** Next.js (App Router)
- **Backend:** Python APIs where needed
- **Database:** Supabase/Postgres
- **Deployment:** Vercel
- **AI:** MiniMax, Anthropic, Google Gemini
